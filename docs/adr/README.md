# Architecture Decision Records (ADR)

Ce répertoire contient les décisions d'architecture du projet, documentées au format ADR.

## Format

Chaque ADR suit la structure :
- **Statut** : Proposé / Accepté / Déprécié
- **Contexte** : Problème ou situation à l'origine de la décision
- **Décisions** : Choix techniques retenus
- **Conséquences** : Impacts positifs et négatifs
- **Références** : Liens vers la documentation
